Configuration Widget Demo

How to use

  • Click the gear icon to open the configuration dialog.
  • Toggle overlay buttons on/off using the switches.
  • Activate/deactivate overlays directly from the config panel.
  • A custom section is injected to demonstrate the addSection API.
