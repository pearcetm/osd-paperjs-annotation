Tool event listener API

Tools extend ToolBase and support addEventListener(eventType, callback) and removeEventListener(eventType, callback). Use the same function reference to remove a listener. Remove listeners when no longer needed (e.g. when a UI component unmounts) to avoid stale callbacks and listener buildup.

Tool vs project events: Tool events (activated, deactivated) are broadcast by the tool instance; subscribe with tool.addEventListener(type, fn) and remove with tool.removeEventListener(type, fn). Project-level events (e.g. ruler-measurement-updated) are emitted on paperScope.project; subscribe with project.on(type, fn) and remove with project.off(type, fn). This demo subscribes to both so the status updates when you switch tools and when you add or edit a measurement.
